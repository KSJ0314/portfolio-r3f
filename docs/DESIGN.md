# 디자인 (DESIGN)

> 아트 방향·레퍼런스·컬러·테마 토큰 정리.

## 아트 방향

**스케치북 위의 그림.** 월드는 빈 종이(모눈종이)이고, 그 위에 필요한 요소만 그려 넣는다.
마을·나무·집·소품 같은 배경 오브젝트는 두지 않는다. 하늘도 없다.

## 뷰 & 카메라

- 직교(Orthographic) 아이소메트릭 뷰 — 원근 왜곡 없는 평면적 항공뷰.
- 카메라는 캐릭터를 화면 중앙에 두고 따라간다(캐릭터 팔로우).
- 방위각은 정확한 대각선(45°)을 피해 살짝 튼다. 고도각은 약 42°.

## 바닥 (모눈종이)

- **손그림 모눈종이.** 흰 종이 + 회색 모눈 선.
- 선은 자로 잰 직선이 아니라 손으로 그은 선이다. 좌우 떨림, 구간별 굵기(필압), 번진 가장자리, 옅은 농담 변화를 준다.
- 굵은 선(5칸마다)은 두지 않는다. 블록으로 구분돼 보여 모눈 느낌을 해친다.
- 텍스처는 심리스로 구워 바닥 전체에 타일링한다. 생성·튜닝 방식은 [DECISIONS 009].
- 바닥은 조명을 받지 않는 재질을 쓴다. 배경은 [DECISIONS 010].
- 이동 범위 경계는 투명 벽처럼 막는다(가장자리 미노출).

## 스테이션 & 배치 구조

- **주제별로 흩어 배치한다.** 중앙(캐릭터 시작점) 주변에 주제(섹션)별로 스테이션을 두는 오픈월드 배치다. 길(도로)은 그리지 않는다. 방향 안내(바닥 화살표·표지판)는 스테이션 실물이 생기는 Phase 8에서 넣는다.
- 섹션·스테이션 수:
  - **About**: 스테이션 3개 — `Intro + Skill` · `Experience + Education` · `Award + Spec(자격증)`
  - **Projects**: 프로젝트마다 스테이션 1개(현재 플레이스홀더 3개, 이후 Firestore 개수대로 생성)
  - **Guestbook**: 스테이션 1개
- **구체적 위치·방향은 임시다.** 스테이션 최종 배치·스케일은 각 스테이션 상세 구현(Phase 8)에서 정한다.
- **스테이션 = 종이 위에 그려진 그림.** 평소에는 2D 그림으로 놓여 있다가, 활성화하면 3D로 전환된다.
- **전환 연출은 스테이션마다 다르게** 만든다. 2D로 남되 카메라를 돌려 3D 맵이 2D로 접히는 느낌을 주거나, 2D 이미지가 애니메이션으로 3D로 일어서거나, 처음부터 3D 오브젝트였다가 펼쳐지는 식이다. 공통 셸·기본 연출은 두지 않는다. ([DECISIONS 007])
- **근접 시 좌클릭 → 제자리 활성화.** 캐릭터가 반경 안에 있을 때만 선택되며(멀어지면 자동 해제), 카메라가 그 오브젝트에 포커스한다.
- 공통층(위치·근접·선택·미니맵)만 프레임워크가 담당하고, **겉모습과 상세 연출은 `스테이션 id → 전용 컴포넌트` 레지스트리로 스테이션마다 독립 구현**한다. ([DECISIONS 006])

## 레퍼런스

- "춘식이의 관찰일기" — 손그림 모눈종이 톤. 다만 그쪽은 모눈이 주된 배경이고, 이쪽은 종이 위에 그림이 그려지고 3D로 살아나는 전환이 시그니처다.

## 컬러 팔레트

| 용도 | 값 | 비고 |
|---|---|---|
| 종이(바닥) | `#fffefc` | 흰 종이 |
| 모눈 선 | `#e0e0e0` | 진하기 상한 0.55로 얹어 실제로는 더 연하게 보인다 |
| 종이 밖 여백 | 라이트 `#ffffff` / 다크 `#14141c` | 하늘이 아니라 여백 |
| UI 배경·표면 | `#ffffff` | 종이와 같은 톤 |
| UI 테두리 | `#e0e0e0` | 모눈 선과 같은 계열 |
| 강조(accent) | 라이트 `#7c9cff` / 다크 `#a98bff` | 캐릭터 임시 색으로도 사용 |
| 섹션 | About `#f0a6ca` · Projects `#8ab6d6` · Guestbook `#a8d5ba` | 미니맵에서 섹션 구분용 |

그 외 색(스테이션 오브젝트·상세 화면)은 **상세 구현 단계에서 스테이션마다 정한다.** 각각 텍스처·에셋·모델을 따로 만들기 때문이다.

## 타이포그래피

- **손글씨 — 감자꽃(Gamja Flower).** 씬 안 글씨(스테이션 라벨·표지판)와 미니맵 라벨.
- **본문 — Pretendard.** 읽어야 하는 콘텐츠(이력서·방명록·상세). 인쇄/PDF도 이쪽.
- 둘 다 self-host. 외부 CDN 요청 없음.
  - 감자꽃 웹폰트: `public/fonts/gamja-flower/` (유니코드 범위별 woff2 조각)
  - 감자꽃 3D 텍스트용: `GamjaFlower-Subset.ttf` (상용 한글 2350자 서브셋, 약 1MB). 3D 텍스트는 웹폰트가 아니라 폰트 파일을 직접 읽는다.
  - Pretendard: npm 패키지의 동적 서브셋 CSS

## 테마 (라이트=낮 / 다크=밤)

- 하늘·스카이박스는 없다. **조명의 색과 밝기만 바뀐다.** 밝은 낮에서 저녁 노을 정도의 변화면 충분하다.
- **네온은 후순위.** 오브젝트 발광이 꼭 들어가지 않아도 된다. 필요하면 가로등 같은 요소로 조명을 켜는 정도로 처리한다.
- 바닥은 조명을 받지 않으므로 낮/밤에 종이색을 곱해 어둡게 한다(임시). 다른 오브젝트는 씬 조명을 받는다. 이 이원 구조는 테마를 제대로 구현할 때 다시 본다.
