## 작성 템플릿

```

- **날짜**: YYYY-MM-DD
- **상태**: 제안 / 채택 / 폐기 / 대체됨
- **맥락**: 어떤 문제·상황이었는가
- **결정**: 무엇을 선택했는가
- **이유**: 왜 그 선택인가 (대안과 트레이드오프)
- **영향**: 이 결정이 가져오는 결과
```
